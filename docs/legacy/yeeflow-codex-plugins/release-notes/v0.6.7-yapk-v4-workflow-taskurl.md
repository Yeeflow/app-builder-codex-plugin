# Yeeflow Builder Plugin v0.6.7

## Scope

v0.6.7 updates the canonical YAPK schema and adds a hard validation gate for workflow assignment task page URL shape.

## Changes

- Replaced `schemas/yapk-schema.json` with the YAPK schema v4 content while keeping the canonical active filename unchanged.
- Mirrored the same schema v4 content into `dist/yeeflow-builder-plugin/schemas/yapk-schema.json`.
- Enforced assignment task page URL shape:
  - `pageurls[].type = 2`
  - `pageurls[].pagetype = 1`
  - `pageurls[].formdef.pagetype = 2`
- Added regression coverage for:
  - `WORKFLOW_TASK_PAGEURL_PAGETYPE_INVALID`
  - `WORKFLOW_TASK_FORMDEF_PAGETYPE_INVALID`
  - `WORKFLOW_TASKURL_PAGE_NOT_FOUND`
  - `WORKFLOW_TASKURL_PAGE_TYPE_INVALID`

## Proof Boundary

This release uses local validation and focused generated smoke artifacts only. No live install, live upgrade, or broad/full app generation was run as part of the release flow.
