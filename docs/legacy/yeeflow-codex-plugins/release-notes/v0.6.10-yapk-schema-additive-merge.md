# Yeeflow Builder Plugin v0.6.10

v0.6.10 prepares an additive merge of the uploaded YAPK schema into the official canonical YAPK schema.

## Included

- Added required-field constraints for `ListGroupInfo`, `MetadataCategoryInfo`, and `MetadataInfo`.
- Preserved `FormNewReports` as the required/current workflow report collection.
- Preserved `FormReports` as optional legacy workflow report storage.
- Preserved the schema v5 `x-yeeflow-standard-additions` block and generated-final hard gates.
- Added source and dist regression coverage for the additive merge.

## Proof Boundary

This release branch was validated locally only. No live install, upgrade, import, or package API call was run. Final tag creation and `stable` promotion remain pending approval.
