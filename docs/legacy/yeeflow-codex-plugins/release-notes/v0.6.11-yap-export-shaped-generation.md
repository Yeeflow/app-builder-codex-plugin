# Yeeflow Builder Plugin v0.6.11

v0.6.11 prepares export-shaped YAP generation hardening for generated-final/import-qualified packages.

## Included

- Dedicated YAP generation contract: `docs/standards/yap-generation-contract.md`.
- Export-shaped YAP generation standard: `docs/standards/yap-export-shaped-application-generation-standard.md`.
- Machine-readable contract companion: `schemas/yap-generation-contract.json`.
- Sanitized first-time generation reference: `docs/studies/normalized/yap-export-shaped-application/reference-shape.normalized.json`.
- Redacted import-qualification study: `docs/studies/yap-export-shaped-generation-import-qualification-study.md`.
- Generated-final validator hard checks for export-shaped metadata, ID shape, ReplaceIds final coverage, decoded-string validation, and proof-boundary reporting.
- Focused regression test: `scripts/test-yap-export-shaped-generation-standard.mjs`.

## Proof Boundary

This release preparation does not update `stable`, move existing tags, push Research, or run live Yeeflow import/install/upgrade API calls. API accepted/queued remains separate from async import completion, app open, form designer hydration, and workflow designer hydration.
