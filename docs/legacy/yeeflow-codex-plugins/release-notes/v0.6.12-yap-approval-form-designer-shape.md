# Yeeflow Builder Plugin v0.6.12

v0.6.12 prepares approval form designer-shape hardening for generated-final YAP packages.

## Included

- Generated-final validation for unique designer-level control IDs across approval form `formdef.children`.
- Native heading/text value validation, including `attrs.headc.title.value` and default-placeholder rejection.
- Designer-safe control-family checks for approval form workspace YAP generation.
- Export-shaped metadata checks for child `ListType`, list/layout metadata, approval DefResource metadata, valid `NoRule`, published form status, and app group ID coverage in `Resource.ReplaceIds`.
- Focused regression test: `scripts/test-yap-approval-form-designer-shape.mjs`.
- Redacted study: `docs/studies/yap-approval-form-designer-shape-study.md`.

## Proof Boundary

API accepted/queued is not import proof. Import complete is not designer hydration proof. Designer opens is not text-render or single-control-selection proof.
